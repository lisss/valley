# The Valley

A Hitchcock-inspired psychological horror quest to find the black heart.

## Quest Overview

This is an interactive web-based puzzle game featuring multiple challenging stages inspired by Alfred Hitchcock's psychological thriller style. Players must solve various puzzles including:

- Caesar cipher decryption
- Literary riddles
- Morse code translation
- Visual observation puzzles
- Reverse text decoding

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to begin the quest.

## Deployment

This project is designed to be deployed on Vercel:

```bash
npm run build
```
